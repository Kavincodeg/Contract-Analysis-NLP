# 📄 Contract Analysis NLP Project

## 🎯 Objective
To extract important information such as **names, organizations, dates, amounts, and durations** from legal contract text using Natural Language Processing (NLP).

---

## 🧠 Concepts Used
- Named Entity Recognition (NER)
- Information Extraction
- Rule-based NLP (Regex)

---

## 🛠️ Technologies Used
- Python  
- spaCy  
- Regular Expressions (Regex)

---

## ⚙️ Features
- Extracts:
  - 👤 Person Names  
  - 🏢 Organizations  
  - 📅 Dates  
  - 💰 Monetary Amounts  
  - ⏳ Contract Durations  
- Removes irrelevant entities (e.g., “the start date”)  
- Eliminates duplicate values  
- Displays structured output
